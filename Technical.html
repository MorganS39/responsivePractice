Introduction

HTML or Hypertext Markup Language is the framework and bones of the internet. This language is responsible for the function and foundation of every web page you have ever viewed. At its core, HTML is a programming language that tells the computer how to display a web page and how it should be structured.

Media

A good way to bring your website to life is through media. These include links to other pages, videos, and images.

Here is a sample of link:

<a href="sample.html">Whatever text you want</a>

Sample video:

<video width="size" height="size" controls>
<source src="movie.mp4" type="video/mp4>
alternative text if video doesn't work
</video>

Sample Image:

<img src="img.jpg" alt="Alt text">
